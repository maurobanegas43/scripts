# Bash Script – Simulación de Ruleta con Método Martingala

Este proyecto contiene un script en **Bash** diseñado para ejecutarse en un entorno **GNU/Linux**, que simula una ruleta de apuestas y aplica el método Martingala para automatizar una estrategia de juego. Es un ejercicio personal para practicar lógica de programación, scripting en consola, y manejo de terminal.

---

## Objetivos del Proyecto

- Desarrollar una interfaz de consola clara y coloreada mediante códigos ANSI para mejorar la experiencia de usuario.  
- Practicar el uso de funciones, bucles, condicionales y generación de números aleatorios en Bash.  
- Implementar la lógica del **método Martingala** (doblar la apuesta tras perder) dentro de un script.  
- Adquirir experiencia en administración de sistemas operativos Linux y en automatización de tareas con scripts de terminal.

---

## Funcionalidades principales

- Generación de un número aleatorio entre 0 y 31 para simular la ruleta.  
- Impresión de la “ruleta” formateada en la consola con colores (verde/rojo/azul/cian).  
- Mecánica de apuesta: el usuario elige “par” o “impar”, realiza una apuesta, y el script aplica la lógica de Martingala.  
- Actualización de saldo tras cada apuesta, hasta que el usuario decide retirarse o se queda sin saldo.  
- Interfaz con menú de opciones (`-n` para jugar normal, `-h` para ayuda).
